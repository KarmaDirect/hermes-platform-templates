# Skill — posts-sociaux

Maintient la présence sociale du client sans qu'il y pense.

## Sources de contenu
- Photos "après" des chantiers terminés.
- Actus injectées par le patron (vocal Telegram, photo).
- Calendrier marketing vertical (ex: BTP — saisonnalité travaux).

## Plateformes
Instagram, Facebook, LinkedIn — via API **Zernio** (~25× moins cher qu'Instagram Graph + Postiz).

## Cadence
Lundi / Mercredi / Vendredi 10h, max 3 posts/semaine.

## Human-in-the-loop
**Oui** — brouillons envoyés au patron pour validation, jamais de publication automatique.
