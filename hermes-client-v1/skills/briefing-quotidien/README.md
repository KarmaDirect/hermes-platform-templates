# Skill — briefing-quotidien

Tous les matins à 7h, le patron reçoit sur Telegram un résumé exploitable en 6 lignes max.

## Sections incluses
- RDV du jour (heure, client, adresse, montant prévu).
- Chantiers actifs et leur statut.
- Factures à relancer aujourd'hui.
- Météo (impact terrain).
- Rappels perso (anniversaires clients, échéances admin).

## Style
Court, sec, percutant — pas un rapport, une checklist verbale.

## Channels requis
`telegram` (le `default_chat_id` du patron est défini lors de l'onboarding).
